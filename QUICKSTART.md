# Quick Start Guide - 1 Minute to Use

## Open & Use

1. **Open `index.html`** in any browser (Safari, Chrome, Firefox, Edge)
2. **Tap "Start Practice"**
3. **Answer questions** - Type or choose from 4 options
4. **Tap "Read Question"** to hear it spoken aloud

That's it! No installation, no setup, works offline.

## Keyboard Shortcuts

- **Enter** - Submit answer when typing
- **Alt+R** - Speak question (depends on browser)

---

## Reset Progress

Want to start over?
1. Open browser DevTools (press **F12**)
2. Go to **Application** tab
3. Click **localStorage**
4. Find `n400_progress` and delete it
5. Refresh page

---

## Troubleshooting

| Problem | Solution |
|---------|----------|
| No sound | Check volume is on, try different browser |
| Progress not saving | Clear browser cache, try again |
| Answer not recognized | Answers are case-insensitive, extra spaces removed |
| App won't open | Make sure all 4 files (index.html, styles.css, app.js, questions-data.js) are in same folder |

---

## Files Needed

All 4 files must be in the same folder:
- `index.html`
- `styles.css`
- `app.js`
- `questions-data.js`

Move them together, open `index.html`, and you're good to go.
